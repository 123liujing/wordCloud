# wordCloud
用python进行文本分词并生成词云
=========================================
## 需要安装的包
分词：jieba 
生成词语：wordcloud
wordcloud依赖于Pillow，numpy，matplotlib 
------------------------------------------
## 其他
分词采用结巴分词，并支持自定义字典
词云图可自己设定背景图（英文词云图不需先进行分词）
------------------------------------------
## 运行结果
运行demo.py,会生成“2017年中央政府工作报告(全文).txt”的词云图，和“词频统计.txt”
